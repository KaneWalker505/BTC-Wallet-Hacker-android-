# BTC-Wallet-Hacker-android-
BTC Wallet Hacker in termux for android, Generates BTC wallets stores priv/pub keys if balance is found (NO API REQUIRED)


## NO API REQUIRED

(will write to hacked.txt when a wallet with a value above 0 is found)


## INSTALL STEPS


1: Download (btchack)


2: Chmod btchack (chmod 777 btchack)


3: run btchack (./btchack)
